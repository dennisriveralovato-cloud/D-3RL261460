# Descripción del proyecto
Infinity-Grid es un dashboard responsivo desarrollado para la empresa ficticia AquaTech, enfocado en el monitoreo en tiempo real de sensores IoT relacionados con la calidad del agua.

El sistema permite visualizar métricas importantes como:
- Nivel de pH
- Oxígeno disuelto
- Temperatura

El diseño fue desarrollado siguiendo la metodología **Mobile First** y adaptándose a distintos dispositivos:
- Smartphones
- Tablets
- Pantallas de escritoriO

# Tecnologías utilizadas
- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
- CSS Sprite
- Checkbox Hack (Menú responsive sin JavaScript)
